NAME
====

**fox** — The Forensic Examiners Swiss Army Knife

SYNOPSIS
========

| **fox** **mcp** \[_flags_ ...]

DESCRIPTION
===========

Loads MCP server and blocks until canceling.

EXAMPLES
========

fox mcp

:   Load the MCP server
