NAME
====

**fox** — The Forensic Examiners Swiss Army Knife

SYNOPSIS
========

| **fox** **mcp** \[_flags_ ...] \[_port_]

DESCRIPTION
===========

Inits MCP server and blocks until canceling.

POSITIONAL ARGUMENTS
====================

The port number (default: **3000**).

EXAMPLES
========

fox mcp 8080

:   Init the MCP server
